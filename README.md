# Anurag Yadav | Portfolio & Chatbot
A modern, responsive portfolio showcasing my journey as an aspiring AI/ML developer and Linux enthusiast. The site features an interactive terminal-based chatbot designed to provide real-time information about my skills, education, and projects.

**Live Site:** [anurag-yadav.in](https://anurag-yadav.in)  
**Repo:** [github.com/anuragyadav0311/portfolio](https://github.com/anuragyadav0311/portfolio)

##  Features
- **Modern UI/UX:** Clean and professional design highlighting my background at VIT Bhopal.
- **Interactive Terminal Chatbot:** A dedicated terminal component where visitors can "query" my profile via a chatbot interface.
- **Dynamic Content:** Showcases my transition from schooling at DPS Risali to becoming an aspiring AI/ML developer.

##  Tech Stack
- **Frontend:** React 18, Vite, CSS3
- **Bot Logic:** Custom JavaScript terminal interface
- **Deployment:** GitHub Pages with Custom Domain Mapping

##  Project Structure
- `src/components/Terminal`: The core chatbot component.
- `src/sections`: Main portfolio content (About, Education, Skills).
- `public/`: Assets including resume and project snapshots.
