Windows 11-Inspired Portfolio Rebuild
Goal
Completely rebuild the Portfolio-windows/ portfolio from its current retro DOS/Win95 style into a modern Windows 11-inspired desktop experience, taking design cues from kassq.dev, vovacodes.ca, and portfoli-os.vercel.app.

Key Design Decisions
Visual Direction: Windows 11 (Modern, not Retro)
Acrylic/Mica translucent window backgrounds (frosted glass effect)
Rounded corners everywhere (8–12px border-radius)
Centered taskbar with hover scale effects (like Win11 dock)
Segoe UI-inspired font (using Inter as web alternative)
Dark theme by default with a subtle gradient desktop wallpaper (Win11 "Bloom" inspired)
Smooth animations — window open/close scale transitions, taskbar icon bounces
Interactive Features
Boot Sequence — Modernized: Windows 11-style loading spinner + "AY" branding (not DOS text)
Desktop Icons — Left-aligned grid with emoji/icon + label, double-click to open
Draggable + Resizable Windows — With snap preview indicators, maximize/minimize/close (traffic-light style or Win11 icons)
Centered Taskbar — Frosted glass, pinned app icons with tooltip labels, active app indicator dot, system tray (clock, wifi icon, battery)
Start Menu — Win11-style floating panel: search bar at top, pinned apps grid, recommended section (recent projects), user profile at bottom
Right-click Context Menu — Desktop right-click shows options like "Refresh", "Personalize", "About this PC"
Lock Screen — On initial load before boot, shows time + date + "Click to unlock"
